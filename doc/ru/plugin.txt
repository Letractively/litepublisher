Руководство как создать свой  плагин

Плагин в LitePublisher - это обычный PHP класс, на который не накладывается никаких формальных ограничений. При инсталяции класс будет добавлен коллекцию классов LitePublisher. Чтобы плагин появился в списке плагинов в админке на странице плагинов следует сделать следующее:

- создать папку впапке plugins, желательно выбрать описательное имя папки. В этой папке будут находится все файлы плагина.
- в папке плагина создать файл about.ini, в котором описать плагин. После этого плагин появится в списке плагинов в админке. Заметим, что появится в списке вовсе еще не означает, что вы уже создали плагин.

Таким образом файл about.ini является точкой входа в плагин. Опишем формат файла about.ini для плагинов (Следует заметить что файлы about.ini тем похожи на файлы плагинов). Файл about.ini должен иметь кодировку utf8, если в нем есть русский текст. Обязательной является наличие секции [about] в about.ini. В секции [about] есть следующие параметры:

- classname = tmyclass; имя PHP класса плагина, обязательный параметр
- filename = "mypllugin.php"; имя файла в папке плагина, где находится класс плагина classname, обязательный параметр
- adminclassname = ""; имя класса админки плагина, если нет админки у плагина, то пустая строка. Обязательный параметр.
- adminfilename = ""; имя файлав папке плагина, где находится класс adminclassname, может быть пустой строкой. Обязательный параметр
- version = "1.0"; версия плагина, может быть любой и показывается в админке, обязательный параметр
- author = "Vladimir Yushko"; имя автора плагина, например ваше имя
- url = "http://litepublisher.com/"; адрес сайта разработчика плагина, показывается в админке как  ссылка на сайт разработчика
- name = "My super plugin"; имя плагина в админке
- description = "Some description"; описание плагина в админке

По умолчанию принято, что в секции [about] весь текст на английском. В файле about.ini можно размещать неограниченное количество языковых секций, например для русскго языка секция [ru],:

- author = "Владимир Юшко"; имя автора плагина на русском
- url = "http://litepublisher.ru/"; сайт автора, может отличаться
- name = "Мой супер плагин"; название плагина для админки
- description = "Какое то описание плагина"; какое описание плагина для админки

Языковая секция перекрывает секцию [about[. Если нет языковой секции, то используется секция [about]. Чтобы было проще понимать разберем устройство плагина на примере простого плагина shortcode - короткие коды.
