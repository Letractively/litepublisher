classname = tstorage 
parent = tfilestorage

Класс только со статическими методами наследник от [[tfilestorage]]. Глобальное хранилище данных. Данные загружаются перед работой всех скриптов и сохраняются только один раз по окончании работы. Рекомендуется для часто меняющихся данных. Исползуется в классах [[toptions]], [[tsite]], [[ttemplate]]. Также исползуется всеми классми БД версии для хранения своих событий и некоторых свойств. Не рекомендуется для редко используемых или больших по объему данных.

method = save
type = boolean
access = public

Статический метод перекрывает родительский метод  [[tfilestorage::save]] для сохранения объектов в глобальное хранилище.

method = load
type = boolean
access = public
Статический метод перекрывает родительский метод [[tfilestorage::load]] и восстанавливает данные объекта из глобального хранилища.


method = savemodified
type = boolean
access = public

Статический метод сохраняет данные хранилища, если они были изменены. Если ни один из объектов не сохранял данные в хранилище, то ничего не происходит.

method = loaddata
type = boolean
access = public

Статический метод загружает данные самого хранилища. Вызывается один раз.
