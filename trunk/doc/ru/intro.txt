LitePublisher - это система управления контентом (CMS), сделанная на PHP с использованием MySQL (есть версия только на файлах). Отличительной особенностью движка являются высокая скорость и низкое потребление памяти при огромных возможностях. Вы можете при помощи LitePublisher создавать сайты сильно отличающиеся по функционалу. Приведем примеры решений на LitePublisher не требующих дополнительных затрат или разработки.

[html]
<dl>
<dd>Блог</dd>
<dt>
Сразу после инсталляции по умолчанию получаем классический блог с постами, комментариями, RSS, рубриками и метками и остальным нужным для блога. В дистрибутиве есть плагин кросспостинга в livejournal.
</dt>
<dd>Новостной сайт</dd>
<dt>
В дистрибутиве есть плагин пользовательских новостей - посетители смогут зарегистрироваться и публиковать новости. Есть плагин голосований. Можно управлять пользователями и их правами. Есть плагины популярных визуальных редакторов ckedit и tinymce, позволяющие красиво оформить текст записей.
</dt>
<dd>Фотосайт</dd>
<dt>
По умолчанию в LitePublisher уже встроена галерея картинок и флешевый загрузчик файлов. В дистрибутиве есть 3 плагина слайдшоу. Для фотохудожника ничего не придется настраивать - уже все работает. Остается только загружать фотки, а остальное сделает движок: создаст миниатюры и организует их в галерею.
</dt>
<dd>Музыкальный сайт</dd>
<dt>Для увлеченных музыкой в LitePublisher встроены аудио и видео плееры - просто загружайте файлы, чтобы они проигрывались на ваших страничках. Есть плагины интеграции с youtube.com. 
</dt>
<dd>Управление PHP проектом</dd>
<dt>
В дистрибутиве есть плагины тикет системы, wiki, документирование PHP кода, обзор исходников, создание FAQ. Мощный набор уникальных инструментов в одном дистрибутиве. Пару кликов и все это уже подключено к сайту.
</dt>
</dl>
[/html]

Мы перечислили самые простые и готовые решения на базе LitePublisher, возможно вы сможете для себя собрать сайт применяя следующие основные функции LitePublisher:
